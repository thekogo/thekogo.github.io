CSS systax ประกอบด้วยอะไรบ้าง

Selector
- เป็นส่วนสำหรับการเลือกระส่วนการนำ css ไปใช้งานในส่วนไหน
Declaration
- เป็นส่วนของการประกาศค่าว่าต้องการให้มีลักษณะอย่างไรบ้าง ซึ่งจะแบ่งเป็น Property คือการกำหนดลักษณะ และ value คือค่าที่ต้องการ

Example Syntax

Selector {Property : value }

h1 {color : red}

การเขียน CSS มีกี่แบบอะไรบ้าง

ประเภทของการเขียน ลักษณะ
การเขียนแบบ inline style เป็นการเขียนแทรก css ใน element นั้นผ่านทาง Attribute style
การเขียนแบบ internal file เป็นการเขียน css ไว้ใน Tag <style> </style> ซึ่งจะเรียกใช้งานผ่าน Attribute class
การเขียนแบบ external file เป็นการเขียน css แยกไว้ในอีกไฟล์นึงแล้วทำการ import เข้ามาใช้งานในไฟล์ html ซึ่งจะสามารถเรียกใช้งาน css ได้ผ่าน Attribute class เช่นกัน

selector มีกี่แบบอะไรบ้าง

ประเภท คำอธิบาย ตัวอย่าง
The CSS element Selector ใช้ Tag HTML เป็น selector ดังนั้น เมื่อใช้ Tah HTML นั้น ๆ จะถูก CSS จัดการด้วย p {color: red;}
The CSS id Selector ใช้ Attribute id ในการกำหนด css #paragraph {color : red;}
The CSS class Selector ใช้ Attribute class ในการกำหนด css .paragraph {color : red;}
The CSS Universal Selector กำหนด css ให้ทุก HTML elements * {color : red;}
The CSS Grouping Selector กำหนด css ให้ HTML elements ที่ระบุมี css ตามที่กำหนด p, h1, h2 {color : red;}

Combinator ทีกี่แบบอะไรบ้าง

ประเภท สัญลักษณ์ อธิบาย ตัวอย่าง
Descendant Selector เว้นวรรค เป็นการเลือก element ทั้งหมดที่ตรงกับ selector h1 p {color : red}
Child Selector > เป็นการเลือก element ทั้งหมดที่เป็นลูกของ element เริ่มต้น เช่น Tag p ทั้งหมดที่อยู่ใน div div > p {color : red}
Adjacent Sibling Selector + เป็นการเลือก element ที่อยู่หลังจาก element นั้น ๆ เช่น เลือก Tag p ที่อยู่หลัง div นั้นทันทีเพียงตัวเดียว div + p {color : red}
General Sibling Selector ~ เป็นการเลือก element ที่อยู่หลังจาก element นั้น ๆ ทั้งหมด เช่น เลือก Tag p ที่อยู่หลัง div ทั้งหมด div ~ p {color : red}

property และ value คือ อะไร แตกต่างกันอย่างไร

ชนิด ความหมาย ตัวอย่าง
property เป็นคุณลักษณะที่ต้องการให้เป็น property คือ color
value เป็นหน่วยของคุณลักษณะนั้น value จะเป็นค่าของสี เช่น red

property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง

กลุ่ม ใช้ทำ
content ใช้จัดการกับข้อความต่าง ๆ เช่น สีของข้อความ การจัดกึ่งกลาง
border ใช้จัดการเกี่ยวกับกรอบของ element
image ใช้จัดการเกี่ยวกับรูปภาพ เช่น ขนาดความกว้าง
margin ใช้จัดการเกี่ยวกับ การเว้นระยะห่างนอก border
display ใช้จัดการรูปแบบแสดงผลของ elements
grid เป็นหนึ่งใน display ที่ใช้แสดงลักษณะของตาราง
flex เป็นหนึ่งใน display ที่ใช้จัดการรูปแบบแสดงผลว่ามีช่องว่างระหว่าง element, จัดให้อยู่ตรงกลาง
media queries ใช้กำหนดขนาดการแสดงผลตามขนาดของหน้าขอ Responsive

value unit มีกี่ประเภทอะไรบ้าง แต่ละประเภทคืออะไร

ประเภท คือ
Absolute Lengths เป็นการกำหนดค่าแบบตายตัว เช่น px
Relative Lengths เป็นการกำหนดค่าแบบไม่ตายตัวตามขนาดหน้าจอ เช่น em %

value ใน display property มีอไะรบ้าง

Value Description
inline แสดงเป็น inline element ค่าของ width กับ height จะไม่ส่งผลใด ๆ
block แสดงผลเป็น block element (like <p>). จะขึ้นบรรทัดใหม่เสมอ มีความกว้าง ความยาว
contents Makes the container disappear, making the child elements children of the element the next level up in the DOM
flex แสดงผลเป็น block-level flex container
grid แสดงผลเป็น block-level grid container
inline-block แสดงผลเป็น inline-level block container. คล้ายเป็น inline element แต่สามารถกำหนดความ กว้าง ยาว ได้
inline-flex แสดงผลเป็น inline-level flex container
inline-grid แสดงผลเป็น inline-level grid container
none The element is completely removed

Grid และ Flex ต่างกันอย่างไร

ประเภท คำนิยาม มิติ
Grid Container-Based 2 มิติ
Flexbox Content-Based 1 มิติ

justify and align แตกต่างกันอย่างไร

ประเภท คำอธิบาย
justify เป็นการจัด element แนวนอนด้านข้างว่าจะเว้นลักษณะยังไง
align เป็นการจัด element ในแนวตั้ง
